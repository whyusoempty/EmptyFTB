# EmptyFTB quest translator

AI-переводчик текстов Minecraft-модпаков: квесты FTB Quests, ланги KubeJS, книги
Patchouli. Работает с любым OpenAI- или Anthropic-совместимым API (по умолчанию
[vibecode.moe](https://vibecode.moe), но можно указать OpenAI, Anthropic,
OpenRouter, локальный Ollama/LM Studio — что угодно). Без npm-зависимостей,
нужен только Node.js 18+.

## Запуск

Двойной клик по ярлыку **EmptyFTB - переводчик квестов** на рабочем столе (или
`start-ui.bat` в этой папке) открывает мини-GUI в браузере на
`http://localhost:3210`.

## Настройка API

Вставь ключ в `config.json`:

```json
{
  "apiKey": "vk-...",
  "baseUrl": "https://vibecode.moe/v1",
  "apiFormat": "auto",
  "model": "claude-sonnet-5"
}
```

- `apiFormat: "auto"` сам понимает формат по имени модели: `claude-*` идёт через
  Anthropic-формат (`/v1/messages`, заголовок `x-api-key`), всё остальное —
  через OpenAI-формат (`/chat/completions`, `Authorization: Bearer`). Если у
  провайдера нестандартные имена моделей — задай `"openai"` или `"anthropic"`
  явно.
- Ключ можно вместо файла положить в переменную окружения `API_KEY` (или
  `VIBECODE_API_KEY` для обратной совместимости).
- Любой другой провайдер — просто меняешь `baseUrl` и `apiKey`:
  - OpenAI: `https://api.openai.com/v1`
  - Anthropic: `https://api.anthropic.com/v1`
  - OpenRouter: `https://openrouter.ai/api/v1`
  - Локальный Ollama (OpenAI-совместимый режим): `http://localhost:11434/v1`

В мини-GUI то же самое можно сделать без правки файла — открой «Дополнительные
настройки» и впиши свой Base URL / ключ / формат прямо в форме, это
переопределит `config.json` только для этого прогона и никуда, кроме
локального сервера и выбранного провайдера, не уходит.

## Использование

```powershell
# посмотреть доступные модели
node src/cli.js models

# прикинуть объём и расход токенов без перевода
node src/cli.js translate "C:\путь\к\сборке" --dry

# перевести (язык по умолчанию ru_ru)
node src/cli.js translate "C:\путь\к\сборке" --lang ru_ru --model claude-sonnet-5

# мини-GUI в браузере
node src/cli.js ui
```

Путь можно указывать как корень сборки, так и `config/ftbquests/quests` напрямую.

## Что поддерживается

За один прогон переводится всё, что найдётся в сборке (общий кэш и батчинг):

- **FTB Quests, новый формат** (MC 1.20+): текст в `quests/lang/en_us.snbt` → пишет `quests/lang/ru_ru.snbt` рядом. Оригинал не трогается.
- **FTB Quests, старый формат**: текст прямо в `chapters/*.snbt` → пишет переведённую копию в `quests-translated-ru_ru/`, содержимое потом копируешь поверх оригинала (бэкап сделай сам).
- **KubeJS lang**: `kubejs/assets/<ns>/lang/en_us.json` → `ru_ru.json` рядом (названия кастомных предметов, тултипы и т.п.).
- **Patchouli**: книги в `patchouli_books/` или `config/patchouli_books/` — папка `en_us/` зеркалится в `ru_ru/` (переводятся name/description/landing_text/title/text/subtitle).

Захардкоженный текст в kubejs-скриптах (`.displayName("...")`, `Text.of(...)`) автоматически не правится — тулза его находит и предупреждает; такое лучше выносить в lang-файл.

## Архив для друзей

После перевода в корне сборки появляется `translation-<lang>.zip` — пути внутри
относительны `.minecraft`, так что друзья просто распаковывают его в свою папку
сборки и всё встаёт на места.

## Оценка расхода токенов

`--dry` (или кнопка «Прикинуть объём» в GUI) показывает не только объём текста,
но и прикидку расхода — сколько строк ещё не в кэше, сколько батчей понадобится
и во сколько это встанет на текущей модели и на паре более дешёвых для сравнения
(claude-haiku-4-5, gemini-3-flash-preview, gpt-5.4-mini). Цифры приблизительные
(±15-20%, точного токенизатора у нас нет), откалиброваны по боевым прогонам.

## Батчинг

`batchMaxItems` / `batchMaxChars` в `config.json` (или в разделе
«Дополнительные настройки» в GUI) — сколько строк и сколько символов
суммарно уходит в одном запросе к модели. Больше — меньше запросов и
оверхеда на системный промпт, но выше риск, что модель обрежет или
перепутает порядок в длинном ответе. Дефолт 30 строк / 4000 символов.

## Фичи

- Батчинг + параллельные запросы (настраивается в `config.json` или в GUI).
- Кэш переводов в `.cache/` — повторный прогон бесплатный, упавший можно перезапустить.
- Глоссарий в `glossary.json` — принудительные соответствия терминов.
- Сохраняются коды форматирования Minecraft (`&a`, `§6`), разметка FTB (`{image:...}`) и Patchouli (`$(l)...$()`), айди предметов, названия модов.
- Мини-GUI: FAQ по кнопке, живой список моделей провайдера, выбор/ввод любой модели вручную.
